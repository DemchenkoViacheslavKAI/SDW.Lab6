# Lab6 Java Project

## Опис завдання
Лабораторна робота: «Дослідження фреймворку колекцій Java».

Потрібно розробити застосунок мовою Java, у якому створюється та обробляється
колекція об'єктів згідно з варіантом завдання.

Тип колекції: список цілих чисел, реалізований за допомогою `ArrayList`.
Обробка колекції: створити список заданої розмірності, задати ціле число `N`
в межах розмірності списку та виконати циклічний зсув списку на `N` позицій вправо.

## Що реалізовано
- Реалізовано введення з клавіатури розмірності списку.
- Програма створює колекцію `ArrayList<Integer>`.
- Користувач вводить усі елементи списку з клавіатури.
- Реалізовано введення числа `N` в межах розмірності списку.
- Виконується циклічний зсув списку вправо на `N` позицій за допомогою `Collections.rotate`.
- У консоль виводяться повідомлення про всі основні етапи обробки:
  - створення списку;
  - додавання елементів;
  - введення величини зсуву;
  - нормалізація зсуву;
  - виведення початкового та результуючого списків.
- Додана підтримка UTF-8 виводу для коректного відображення українського тексту в консолі.

## Структура проєкту
- `Main.java` — основна логіка створення, заповнення та циклічного зсуву списку.
- `TASK.md` — умова лабораторної роботи.
- `README.md` — документація проєкту.

## Алгоритм
1. Зчитати з клавіатури розмірність списку.
2. Створити порожній `ArrayList<Integer>`.
3. Зчитати з клавіатури всі цілі числа та додати їх до списку.
4. Зчитати число `N` в межах розмірності списку.
5. Створити копію початкового списку для збереження вихідних даних.
6. Нормалізувати зсув за формулою `N % size`.
7. Виконати циклічний зсув вправо за допомогою `Collections.rotate`.
8. Вивести початковий і результуючий списки.

## Вимоги
- JDK 11+ (або новіше)
- Налаштована змінна `JAVA_HOME`

## Запуск (Windows)
1. Перейдіть у папку проєкту:
```batch
cd "D:\Coding\University\SDW\Lab6"
```

2. Для коректного відображення українського тексту встановіть UTF-8 кодування консолі:
```batch
chcp 65001
```

Або для PowerShell:
```powershell
chcp 65001
$OutputEncoding = [Console]::OutputEncoding = [System.Text.Encoding]::UTF8
```

3. Скомпілюйте програму:
```batch
javac Main.java
```

4. Запустіть програму:
```batch
java Main
```

5. Введіть:
- розмірність списку;
- елементи списку;
- число `N` для циклічного зсуву вправо.

## Крайні випадки
- Якщо розмірність списку не є додатним цілим числом, програма повторює запит введення.
- Якщо елемент списку не є цілим числом, програма просить ввести його повторно.
- Якщо `N` не є цілим числом, програма просить ввести значення повторно.
- Якщо `N` виходить за межі від `0` до розміру списку, програма повторює запит введення.
- Якщо `N` дорівнює `0` або розміру списку, результуючий список збігається з початковим.
